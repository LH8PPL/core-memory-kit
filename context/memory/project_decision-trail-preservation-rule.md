When revising architecture or decisions, preserve original sketches by appending revised versions alongside (e.g., §16.50 + §16.50.1) rather than overwriting

**Why:** Preserves decision history and makes changes traceable; enables future readers to understand the reasoning and evolution

**How to apply:** Append new numbered versions with cross-references; never delete or replace original sketches
