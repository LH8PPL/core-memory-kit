- Research for the memory kit's knowledge base must meet a high bar for inclusion
- Exclude marketing-framed research, unverified claims, and credential padding
- Memory-specific claims require isolated ablations (memory on/off) to validate; task-performance benchmarks alone don't isolate memory contribution
- Criterion: the research must demonstrate *memory subsystem alone* improved outcomes, not a better harness, prompt, or model

**Why:** The SOURCES artifact must remain credible and actionable. Unverified claims dilute the research base and mislead future development decisions.

**How to apply:** When evaluating new research, ask: does it ablate memory contribution in isolation? Does it avoid credential padding or lazy framing? Only include if memory impact is clearly isolated and evidenced.
