When filing new ideas or findings: lead with the strongest counter-argument, not the pitch. For novel ideas, ask first: "why hasn't anyone done this?" and articulate candidate blockers (hard to measure, not actionable, demoralizing, circular logic). Only after stating the counter-case should you add the pitch.

**Why:** Three recent task entries were filed by assuming flattering explanations instead of interrogating them. Entries filed this way rot until re-examined.

**How to apply:** Draft counter-arguments first. Strong counter-cases make pitches credible; weak or missing ones signal the entry needs more research before filing.
