The user prefers to receive design proposals with 3 concrete options/forks, then picks the preferred approach. They call this "grill-style."

**Why:** This accelerates decision-making and ensures all options are explored before committing.

**How to apply:** When reaching design forks, present 3 concrete options for the user to grill/pick, rather than asking for direction.
